# Camuflagem
Script de estudo de uma ferramenta grafica para uso do steghide para ocultar arquivos.



## Como usar
Extraia o script e altere a permissão com o comando:


```bash
  chmod +x steghide-script-v0.01.sh
```

Execute o script:


```bash
  ./steghide-script-v0.01.sh

```
